# Contributing to jjigit

Thank you for your interest in contributing to **jjigit**!  
This project aims to build a real-time polling and debate platform, with an AI module responsible for generating discussion topics using lightweight LLMs such as Gemma 2B.

This guide describes how to contribute specifically to the AI module and how to maintain a consistent workflow across the project.
